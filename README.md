# go-gui
Go REST API backend served with HTML + Jquery frontend 

  
# Installation
``` 
cd $HOME
git clone https://github.com/vinodborole/go-gui.git

cd $HOME/go-gui/scripts
sh build.sh

```
# Execute

```
cd $HOME/go-gui/bin
./app

```
